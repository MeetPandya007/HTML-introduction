HTML stands for HyperText Markup Language

There are several tags used in order to create HTML site

  • "h1" to "h6" tags are used to set the text size, where h1 is the biggest heading size and h6 is the smallest one.
  • "center" is used to align text, image, video etc in the center.
  • "hr" used for horizontal rule.
  • "br" used to break line and start from new line.
  • "b" tag is used to bold the text.
  • "i" tag is used to italic the text.
  • "u" tag is used to underline the text.
  • "p" tag is used to write in paragraph.
  • "a" tag is known as anchor tag, by which we can redirect to other webpage.
  • "img" tag is used to display image.
  • "table" tag is used to create a table.
  • "tr" tag is used for table row.
  • "td" tag is used for table data.
  • "thead" tag is used for table head.
  • "tbody" tag is used for table body.
  • "tfoot" tag is used for table footer.
  • "label" tag is used for write a label in form.
  • "input" tag is used to take input from the user in form.
  • "audio" tag is used to include audio file.
  • "video" tag is used to include video.
  • "track" tag is used to include captio in video.

CSS: Cascading Style Sheet

There are three types of CSS

  1. Inline CSS: it is used inside the tag, hence it can only be applied on that particular single tag.
  2. Internal CSS: it is used inside the whole html page.
  3. External CSS: Individual CSS file will be created and can be used in html.
